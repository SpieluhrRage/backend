О сервисе погоды

Это вторая статическая страница. Здесь можно разместить краткое описание сайта, для чего он нужен и как им пользоваться.

Наш небольшой сервис показывает статическую информацию о погоде и динамические данные, которые загружаются из базы данных. Так можно продемонстрировать работу связки nginx + Apache + PHP + MySQL.

  • Статические страницы с описанием и иллюстрациями.
  • Динамические страницы с данными из базы (город, температура, состояние).
  • Админ-панель с авторизацией через базу пользователей.
